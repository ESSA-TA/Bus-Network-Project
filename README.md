# Bus Network Project - Premium Transit 🚍

[cite_start]This is an advanced Django-based web application for managing bus routes and bookings, built as an extension of the airport example[cite: 5, 6].

## Features
- [cite_start]**Route Management**: View all available routes[cite: 31].
- [cite_start]**User Authentication**: Full register, login, and logout functionality using Django's built-in User model[cite: 12, 17].
- [cite_start]**Booking System**: Logged-in users can book or unbook seats via POST requests[cite: 33, 34, 38].
- [cite_start]**Route Details**: View specific route information and the list of passengers[cite: 32].

## Bonus Features (+15% points) 🌟
[cite_start]I have implemented all the bonus requirements listed in the lab[cite: 42]:
- [cite_start]**My Routes Page**: A dedicated page showing all routes booked by the logged-in user[cite: 43].
- [cite_start]**Station Detail Page**: Lists all departing and arriving routes for a specific station[cite: 44].
- [cite_start]**Search & Filter**: Users can search for routes by origin or destination[cite: 45].
- [cite_start]**Passenger Count**: Displaying the number of passengers next to each route on the index page[cite: 46].
- [cite_start]**Polished UI**: Styled with Bootstrap for a professional and responsive look[cite: 47].

## Technologies Used
- [cite_start]**Backend**: Django[cite: 14].
- [cite_start]**Database**: PostgreSQL (Production)[cite: 41, 106].
- [cite_start]**Static Files**: WhiteNoise[cite: 41, 145].
- [cite_start]**Deployment**: Live on Railway[cite: 40].

## How to Use
1. Visit the live URL.
2. [cite_start]Register a new account[cite: 27].
3. Browse routes, use the search filter, or visit station details.
4. [cite_start]Log in to book/unbook seats on your favorite routes[cite: 33, 34].
