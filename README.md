# Bus Network Project - Premium Transit 🚍

This is a Django-based web application for managing bus routes and bookings. 
[cite_start]Developed as part of the CS50 Web Lab[cite: 1].

## Features
- [cite_start]**Route Management**: View all available bus routes from the index page[cite: 31].
- [cite_start]**User Authentication**: Full register, login, and logout functionality[cite: 27, 28].
- [cite_start]**Booking System**: Logged-in users can book and unbook seats on specific routes[cite: 33, 34].
- [cite_start]**Admin Panel**: Managed through Django's admin interface for easy data entry[cite: 25].

## Technologies Used
- [cite_start]**Backend**: Django[cite: 17].
- [cite_start]**Database**: PostgreSQL (Production) / SQLite (Local)[cite: 41, 138].
- [cite_start]**Styling**: Bootstrap for a polished UI[cite: 47].
- [cite_start]**Deployment**: Deployed on Railway/Render[cite: 19, 40].

## How to Use
1. Visit the live URL.
2. Register a new account.
3. Browse the available routes on the home page.
4. Click on a route to see details and book your seat.
